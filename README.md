
# Real Time Chat App

This app is still in the making.

## Scripts you can run:
To start the frontend:

```console
foo@bar:~$ cd ./client
foo@bar:~$ npm install
foo@bar:~$ npm start
```
To start the backend:

```console
foo@bar:~$ cd ./server
foo@bar:~$ npm install
foo@bar:~$ npm start
```

## Tech used:
### Backend:

1. Express
2. Cors
3. Socket.io
4. jsonwebtokens
5. bcrypt

### Frontend:

1. React.js
2. socket.io-client
3. react-jwt
4. Tailwind.css
5. Flowbite
